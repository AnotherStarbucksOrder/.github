## ☕ 이세계 스타벅스
- '이세계 스타벅스' 높은 연령층의 사용자들도 쉽게 사용할 수 있는 <b>직관성(Intuitiveness)' 을 갖춘 키오스크</b>입니다.

- 제작 기간 : 2024. 10. 01 ~ 2024. 11. 13
- 주요 기능
    - USER

        - 주문 및 결제
        - 포인트 적립 및 사용
    - MANAGER

        - 로그인 및 로그아웃
        - 메뉴관리
        - 매출관리
        - 주문관리
        - 회원관리
